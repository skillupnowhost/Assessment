# Media Assets

Place a short, optimized `nexus-loop.webm` file here to enable the hero video playback.

The page already includes an animated liquid fallback, so the site remains polished if the video is not present.

Recommended production settings:

- 8-12 seconds
- Muted abstract QA or data-visualization loop
- 1080p or smaller
- WebM first, optional MP4 fallback
- Keep the final asset under 3 MB when possible
